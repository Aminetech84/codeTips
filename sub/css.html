FLEXBOX

Flex container *parent element*

/* Main properties for flex container*/
  • justify-content: Distributes space between and around items or put content to the: right, center or left of the page or change space between elements *always on the main axis*
  • align-items: align items based on height *the element needs a height for it to work*
  • flex-wrap: add rows when width is smaller than the total width of the elements width.
  • flex-direction: Defines the main axis and direction (e.g., row, column).
  • flex-wrap: Allows items to wrap to new lines (nowrap, wrap, wrap-reverse).
  • align-items: Aligns items along the cross axis.
  • Cannot have a max-width on child of an element with max-width.
  • Tips:
    • Use "%" for containers to adopt the free space.
    • Use rem/em for elements to hold their size and flex wrap can work on them.
    • Use "%" then use wrap and fixed w/h in media queries.
  • EM follows the font-size
  • percentages are great for responsive design, and rem/em units help maintain scalability for your flex items.
Example
    
      /* Flex Container (Parent Element) */
      .flex container {
        display: flex; /* Establishes a flex container */

        /* Main properties for flex container*/
        justify-content: space-evenly; /* Aligns flex items along the main axis */
        align-items: center; /* Aligns flex items along the cross axis */
                  

                    align-items: center; /* Aligns flex items along the cross axis */
                    
The align-items property aligns items on the cross axis. For example, if you have a flex container with display: flex and a vertical direction, this property will control the alignment of items vertically within it.

                    
                  

        /* Optional properties for flex container*/
        flex-direction: row;/* Defines the direction of flex items (row, column, etc.) */
        flex-wrap: wrap; /* Allows flex items to wrap onto multiple lines */
        flex-flow: column; /* Shorthand for flex-direction and flex-wrap */
        align-content: stretch; /* Aligns lines of flex items within the flex container */
      }

Flex container *Flex Item (Child Element)*

/* Main properties for flex item*/
Here are some tips:
  • flex child can also be a flex container
  • flex-grow: Controls the proportion of available space taken up by the item when the container is larger than the combined items, works only on small content, flex-grow works better when flex-basis: 0;
  • flex-shrink: Determines how an item will shrink relative to the rest of the items in the container when there isn't enough space.
  • flex-basis: Sets the initial size of an item before the remaining space is distributed, can be set on container or item.
  • order: Changes the visual order of the item.
  • align-self: Allows the default alignment to be overridden for individual flex items.
Example
    
      .flex child {
        /* Optional properties for flex item *child of flex* */

        align-self: center; /* Allows a single flex item to align itself independently */
        
        order: 2; /* Changes the order of the flex item */

        flex-basis: auto; /*default size "width" before applying "flex-grow" or "flex-shrink" */

        flex-grow: 2; /*grow based on flex-basis */ 

        flex-shrink: 2;  /*shrink based on flex-basis */

        flex: 1 0 5em; /* flex: flex-grow flex-shrink flex-basis *Shorthand* */

      }
